# 4.编程的原料 - 变量和函数
## (1) 变量

变量（Variables）很容易理解，它们可以理解成指向存储在内存中的值的标识
```text
>>> a = 1   # 创建一个变量，成为a，它指向1
>>> b = 2   # 创建另一个变量b
>>> a       # 获取a这个标识指向存储在内存中的值
1
>>> b
2
>>>
```

上面的过程可以画成以下图形

![](/assets/img/blogs/pythonContents/chapter02_variable_0.png)

在我们赋予变量后我们也可以重新设置该变量

```text
>>> a = 2    # 将本来指向1的变量指向2
>>> a
2
>>>
```
